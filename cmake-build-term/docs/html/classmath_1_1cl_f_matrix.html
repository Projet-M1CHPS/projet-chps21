Detailed Description

A float matrix stored in a cl::Buffer. Provides wrappers for most clblast operations.

As opposed to math::FloatMatrix, multiple matrices can share the same cl::Buffer, being views of the same data. The memory is freed when the last view is destroyed.

This class uses the default platform/context, and should therefore only be used after clWrapper::initOpenCL() has been called..
