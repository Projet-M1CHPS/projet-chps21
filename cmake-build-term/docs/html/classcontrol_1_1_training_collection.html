◆ alterTensors()

void control::TrainingCollection::alterTensors ( size_t  new_tensor_size )
inline

Alter the location of every input in both sets to create new tensors of the given size. Residual elements will be grouped in a smaller tensors at the end of the lists Note that this is a really costly operation, so it should be used with care.

New tensors will be created with the given size, and the inputs will be moved (Maintaining the same ordering) to fill them up. Older tensors are deleted, and the last tensor might be smaller if the number of inputs is not a multiple of the new size.

Parameters
new_tensor_size The new size for the tensors
